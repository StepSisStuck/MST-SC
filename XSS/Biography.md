# How to get an a alert 

## 1st Example
```html
https://samantha.iscool.com/about?img=" onerror="alert('ethan_was_here')"
```

Explanation: This URL contains an `onerror` attribute that triggers an alert when the image fails to load. This can be used to execute arbitrary JavaScript code in the context of the user's browser.

Impact: An attacker can use this technique to inject malicious scripts into a website, potentially leading to data theft, account compromise, or other security issues.
